# automatic-rotary-phone

Super simple fun phone app for GitHub.

Tiny joke built in:
- **Web 4 beta** = humans and AI can call each other
- Try `404` or `444`

## Run

Just open `index.html` in your browser.

```bash
git init

git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/your-username/automatic-rotary-phone.git
git push -u origin main
```
